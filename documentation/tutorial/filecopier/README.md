# FileCopier 实战教程

> **手写一个带进度条的文件拷贝器**

本教程将带你从零开始，实现一个实用的文件拷贝工具。我们不只是简单地复制文件，还要实现实时进度显示、速度计算、ETA 预估等高级功能。在这个过程中，你会学到文件 I/O、时间处理、终端控制等实用技巧。

## 你将学到什么

完成本教程后，你将：

- 理解分块读写的原理和必要性
- 掌握 C++17 文件系统库 (`std::filesystem`) 的使用
- 学会终端进度条的实现技巧
- 了解时间测量和性能计算的方法
- 能够编写带实时反馈的命令行工具

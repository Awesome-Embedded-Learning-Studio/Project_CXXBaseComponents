# ArgParser 实战教程

> **从零实现一个现代C++命令行参数解析器**

本教程将带你从零开始，手写一个轻量级但功能完整的命令行参数解析器。我们不依赖第三方库，只用现代C++的标准库，让你真正理解参数解析的底层机制。
